🎵 Music Database Project (Backend)
Данный репозиторий содержит реализацию серверной части (Core) музыкальной базы данных на языке C и ресурсы для графического интерфейса.

📂 Структура проекта
c_core/ — Основная логика на языке C.

include/record.h — Описание структуры Song (100% совместимость с GUI).

include/file_handler.h — Протоколы функций для работы с файлами.

src/file_handler.c — Реализация загрузки, сохранения и создания бэкапов.

src/record.c — Вспомогательный файл структуры.

src/sort.c, src/binary_search.c — Заготовки для алгоритмов (задача Человека №2).

CMakeLists.txt — Скрипт для сборки библиотеки libcore.a.

data/ — Бинарные данные.

music_db.dat — Основная база (22 записи).

generate_test_data.c — Утилита для пересоздания базы.

covers/ — Обложки альбомов (используются в GUI).

cpp_gui/ — Исходный код графического интерфейса на C++ (ImGui).

🛠 Компиляция и запуск
1. Сборка Backend библиотеки
Для того чтобы алгоритмы были доступны в GUI, их нужно собрать в статическую библиотеку:

Bash
cd c_core
mkdir build && cd build
cmake ..
make
После этого в папке build появится файл libcore.a.

2. Генерация тестовых данных
Если нужно обновить базу данных или добавить новые песни:

Bash
gcc data/generate_test_data.c -I./c_core/include -o data_gen
./data_gen
3. Сборка GUI
Для компиляции интерфейса необходимо линковаться с собранной библиотекой:

Bash
g++ cpp_gui/main.cpp -L./c_core/build -lcore -lGL -lSDL2 -lGLEW -o music_app
